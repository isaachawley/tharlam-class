local development

  1. download GitHub Desktop
  2. Install it
  3. Sign in with your github account
  4. "Clone" your repository and REMEMBER WHERE YOU PUT IT
  5. Create a web page in Word with some pictures
  6. export the page to your repository directory
    1. you can view and edit your page locally in that location
    2. open it in Chrome
  7. in Github Desktop, "Commit to main" and "Push to origin"
